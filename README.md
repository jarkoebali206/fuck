##  Инструменты

- **pylint** — глубокий анализ кода, проверка стиля и логики (оценка 0-10)
- **flake8** — проверка стиля по PEP 8 (пробелы, отступы, длина строк)

##  Проверка качества

Оба файла прошли все проверки:
```bash
flake8 shop.py students.py
# Нет вывода = нет ошибок ✓

pylint shop.py students.py
# shop.py: 9.5/10 ✓
# students.py: 9.2/10 ✓
```

##  Сравнение flake8 и pylint

| Критерий | flake8 | pylint |
|----------|--------|--------|
| Проверка стиля PEP 8 | да | да |
| Неиспользуемые переменные | нет | да |
| Документация (docstring) | нет | да |
| Логические ошибки | нет | да |
| Оценка качества | нет | да (0-10) |
| Скорость | Быстрый | Медленнее |

**flake8** удобнее использовать для быстрой проверки стиля в процессе разработки.  
**pylint** — для комплексного анализа качества кода перед коммитом или в CI/CD.
